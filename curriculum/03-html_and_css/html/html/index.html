Accessibility notes

Use role, aria-*, meaningful labels, and proper heading order. Avoid using tabindex unless necessary.

Quick reference

  • Meta: <meta charset>, viewport
  • Favicons: <link rel="icon">
  • Forms: required, type validation, novalidate
  • Security: rel="noopener noreferrer" on external links
  • Performance: preload, loading="lazy"
